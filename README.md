Trouble Shooting


**The file downloaded but nothing happens when I open it**

Make sure you are opening the file in a web browser, not a text editor. Right click the file, select Open With, and choose Chrome or Edge. The file will not run on its own by double clicking in some systems depending on your default program settings.




**The color picker button does nothing when I click it**

The color picker uses a browser feature called the EyeDropper API. This only works in Chrome and Edge. If you are using Firefox, Safari, or another browser, the Pick Color button will not appear and a manual hex input field will show instead. Type a hex color code like #ff5733 into that field and click Add.


**The color picker opens but then closes without picking a color**

This happens if you press Escape or click somewhere that cancels the picker before selecting a color. Just click the Pick Color button again and click directly on the color you want to sample.


**My notes disappeared**

The tool does not save data between sessions. If you close the tab or refresh the page everything resets. If you need to keep your notes, copy them out before closing. A future version may include export support.


**The layout looks broken or buttons are misaligned**

Try zooming your browser back to 100 percent. Press Ctrl and 0 at the same time to reset zoom. The tool is designed for standard desktop zoom levels.

**The sidebar is missing**

The sidebar may be collapsed. Look for a small arrow button on the far left edge of the screen and click it to expand the sidebar back out.

**The reference panel is not opening**

Click one of the buttons in the sidebar such as Visual Quality, Instruction Following, or Correctness. If the sidebar is collapsed, expand it first using the arrow button on the left edge.

**Dark mode is not saving between sessions**

The tool saves your theme preference using browser local storage. If you are using private or incognito mode, local storage is cleared when you close the window. Open the file in a regular browser window for the preference to stick.

**The file looks like plain code when I open it**

Your computer opened the file in a text editor instead of a browser. Right click the file, choose Open With, and select Chrome or Edge from the list.

**Something stopped working after I edited the file**

The entire app lives in one HTML file. If you made changes and something broke, compare your version to the original download. Even a missing quote or bracket can cause issues. It is a good idea to keep a backup copy before making any edits.
